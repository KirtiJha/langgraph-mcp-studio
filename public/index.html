Frequently Asked Questions

Everything you need to know about MCP Studio Enterprise Platform

What is MCP Studio?

MCP Studio is an enterprise desktop platform for managing Model Context Protocol (MCP) servers, discovering 1000+ public APIs, and seamlessly integrating them with AI assistants. Built with React 19, TypeScript, and Electron for enterprise-grade cross-platform compatibility.

How does enterprise API conversion work?

MCP Studio automatically analyzes OpenAPI 3.0+ specifications and generates production-ready TypeScript MCP servers with enterprise features including parameter validation, OAuth2 authentication, rate limiting, error management, and monitoring. APIs become immediately available as tools for AI assistants with full enterprise security.

What enterprise authentication providers are supported?

MCP Studio supports enterprise OAuth2 providers including Google Workspace, GitHub Enterprise, Microsoft Azure AD, IBM Cloud SSO, and custom enterprise identity providers. Configure multi-factor authentication, SAML integration, and role-based access control for secure enterprise deployment.

What's the enterprise licensing model?

MCP Studio is 100% open source and free for all enterprise use. Download the desktop application, deploy with Docker, build from source, or contribute to the project. Enterprise support and consulting services available separately for large-scale deployments.

What enterprise platforms are supported?

MCP Studio supports macOS 11.0+ (Intel & Apple Silicon), Windows 10/11 Enterprise, and Linux distributions. Built with Electron for consistent enterprise deployment across platforms with centralized management, auto-updates, and enterprise security compliance.
