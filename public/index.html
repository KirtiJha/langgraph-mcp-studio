Frequently Asked Questions

Everything you need to know about MCP Studio

What is MCP Studio?

MCP Studio is a modern desktop application that helps you manage Model Context Protocol (MCP) servers, discover public APIs, and integrate them with AI assistants. It's built with React, TypeScript, and Electron for cross-platform compatibility.

How does API conversion work?

MCP Studio automatically analyzes OpenAPI specifications and generates production-ready TypeScript MCP servers. The conversion process includes parameter validation, authentication handling, and error management, making APIs immediately available as tools for AI assistants.

What authentication providers are supported?

MCP Studio supports Google OAuth2, GitHub OAuth2, Microsoft OAuth2, IBM SSO, and custom enterprise SSO providers. You can easily configure authentication for secure access to your MCP servers and APIs.

Is MCP Studio free?

Yes! MCP Studio is 100% open source and free to use. You can download the desktop application, build from source code, or contribute to the project on GitHub.

What platforms are supported?

MCP Studio is available for macOS (10.15 or later) and Windows (10 or later). The application is built with Electron, ensuring consistent performance across platforms.
